# Star Stitcher Frontend

The Next.js 15 App Router frontend application for the Star Stitcher boutique, optimized for mobile customer workflows and desktop admin operations.

## Tech Stack
- Next.js 15 (App Router, Tailwind CSS, TypeScript)
- TanStack React Query (State fetching & mutation cache)
- Zustand (Global client profile state store)
- Axios (HTTP API client)
- React Hook Form & Zod (Form schema validation)

## Getting Started

1. Copy `.env.example` to `.env` and fill in the required parameters:
   ```bash
   cp .env.example .env
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```
